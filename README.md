- **Quản lý giáo viên:** Lịch dạy, thông tin chuyên môn.
- **Quản lý khóa học:** Tạo mới, chỉnh sửa và lộ trình học.

## 📥 HƯỚNG DẪN CÀI ĐẶT & CHẠY PROJECT

### 🔹 Bước 1: Clone source code
```bash
git clone [https://github.com/VanhauDAU/DoAnChuyenNganhCNPM.git](https://github.com/VanhauDAU/DoAnChuyenNganhCNPM.git)
cd DoAnChuyenNganhCNPM
### Bước 2: Cài đặt thư viện PHP
composer install
### Bước 3: Tạo file môi trường
cp .env.example .env
Không commit file .env lên GitHub
### Bước 4: Cấu hình Database trong .env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306        # macOS (XAMPP) có thể là 3307
DB_DATABASE=DoAnChuyenNganhCNPM_TrungTamNN
DB_USERNAME=root
DB_PASSWORD=
Tạo database trống trong MySQL với tên
DoAnChuyenNganhCNPM_TrungTamNN
### Bước 5: Khởi tạo ứng dụng
php artisan key:generate
php artisan migrate
### CHẠY PROJECT
🔸 Cách 1: Laravel built-in server (Khuyến nghị)
php artisan serve
Truy cập: http://127.0.0.1:8000🔸 
🔸 Cách 2: Dùng XAMPP (Apache)
Di chuyển toàn bộ thư mục dự án vào htdocs.
Truy cập: http://localhost/DoAnChuyenNganhCNPM/public
